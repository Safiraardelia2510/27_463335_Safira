Esai

Mata kuliah Sistem Informasi Geografis Berbasis Web terdiri dari teori dan praktikum. Kali ini, saya akan menjelaskan mengenai apa saja yang sudah dipelajari di bagian praktikum. Beberapa hal yang pokok untuk membangun sebuah web sederhana sudah dipelajari, seperti HTML, CSS, dan JS. Praktik menggunakan komponen-komponen tersebut secara sederhana sudah dilakukan. Sebagai contoh, terdapat tugas untuk membuat tulisan tulisan sederhana dengan berbagai style teks bergambar dan juga membuat tabel tabel yang memerlukan sebuah masukan atau input. Selain itu, salah satu hal yang menarik untuk dipelajari adalah mengenai Bootstrap dan Leaflet JS. Bootstrap digunakan untuk mempermudah pembuatan web bahkan tersedia template gratis sehingga dalam membangun web tidak harus dari awal. Leaflet js juga menarik untuk dipelajari dimana dengan leaflet tersebut dapat menampilkan sebuah webmap sederhana disertai pop-up.
